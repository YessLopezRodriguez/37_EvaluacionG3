7) Qué es un arreglo, cuál es su sintaxis y cómo puedes acceder a sus elementos?:

Un arreglo es una colección de datos

Sintaxis: Nombre del arreglo y dentro de [] sus datos separados por comas: array[1,2,];

Accedemos: Con el nombre del arreglo y entre [] la posición que empieza del 0 al n valor Ejemplo: var numeros = [1,2,3]; console.log(numeros[1]);

8) Qué son las funciones y cuál es su sintaxis de declaración y de llamada:

Método que ejecuta acciones y regresa un valor, las funciones permiten reutilizar código

Sintaxis: function nombre_funcion (parametro) { tareas ... return resultado;};

Accedemos: nombre_funcion (valor_parametro);

9) Qué son los objetos, cuál es su sintaxis y cómo puedes acceder a sus propiedades?:

Es un elemento con propiedades y sus correspondientes valores.

Sintaxis: var objeto = { propiedad1: valor1, ... };

Accedemos: objeto.propiedad1;
